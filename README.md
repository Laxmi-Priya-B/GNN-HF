# 🫀 GNN-HF: ECG Heart Failure Prediction

AI-powered system for detecting heart failure from ECG images using Graph Neural Networks.

---

## ⚡ Overview

GNN-HF analyzes ECG images and predicts heart failure risk using a hybrid deep learning approach.
The system integrates a full-stack web application with a machine learning pipeline for real-time predictions.

---

## 🚀 Features

* ECG image-based heart failure prediction
* Hybrid deep learning model (CNN + GNN)
* Real-time analysis and results
* Secure user authentication
* Clean and responsive interface

---

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript, TailwindCSS
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Machine Learning:** PyTorch, OpenCV

---

## ▶️ Getting Started

```bash
git clone https://github.com/your-username/GNN-HF.git
cd GNN-HF

# Backend
cd backend
npm install
node server.js

# Frontend
cd frontend
npm install
npm run dev
```

Open: http://localhost:8080

---

## 🧠 How It Works

1. User uploads an ECG image
2. Backend processes the image
3. ML model analyzes patterns using CNN + GNN
4. System returns prediction and risk assessment

---

## 🎯 Use Case

Designed for assisting early detection of heart failure by analyzing ECG data efficiently and providing quick insights.
